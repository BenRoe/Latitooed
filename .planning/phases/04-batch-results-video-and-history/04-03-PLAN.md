## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Active coordinate state to persistence | User-selected places/manual values become durable recent-coordinate rows. |
| SwiftData store to coordinate UI | Persisted values re-enter the active coordinate selector. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-03-01 | Information Disclosure | recent-coordinate model | mitigate | Persist only label, latitude, longitude, and timestamp; no media paths or result records. |
| T-04-03-02 | Tampering | persisted coordinate reuse | mitigate | Rehydrate through `CoordinateSelection` validation before updating active selection. |
| T-04-03-03 | Reliability | SwiftData writes | mitigate | Use explicit `modelContext.save()` after correctness-sensitive mutations, optionally gated by `hasChanges`. |
| T-04-03-SC | Scope creep | favorites/custom names | mitigate | Do not add pinned favorites or custom naming in Phase 4. |
</threat_model>
