## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User-selected file metadata to UI state | Display names, folders, result messages, and diagnostics come from local files and helper output. |
| Selection state to write action | The current selection is for review only; Apply Location still writes the loaded file set. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01-01 | Information Disclosure | multi-selection summary | mitigate | Show aggregate counts only for multiple selection; do not expose extra paths or diagnostics from arbitrary selected files. |
| T-06-01-02 | Spoofing | selected-file review | mitigate | Derive review state from `selectedFiles` and `selectedFileIDs`; ignore stale IDs not present in current files. |
| T-06-01-03 | Tampering | persisted preferences | mitigate | Do not persist table/grid mode in SwiftData, UserDefaults, or bookmarks. |
| T-06-01-SC | Supply chain | dependencies | accept | No third-party package or framework is introduced. |
</threat_model>
