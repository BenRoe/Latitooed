In the grid-mode header `HStack` (the one containing `Text("Loaded Files")` and the existing table/grid view-mode picker), add a second segmented `Picker` for thumbnail size, visible only when `viewModel.selectedLoadedFilesViewMode == .grid`. Picker:
```swift
if viewModel.selectedLoadedFilesViewMode == .grid {
    Picker("Thumbnail size", selection: $thumbnailSize) {
        ForEach(ThumbnailSize.allCases) { size in
            Text(size.displayName).tag(size)
        }
    }
    .labelsHidden()
    .pickerStyle(.segmented)
    .frame(maxWidth: 180)
}
```

Pass `thumbnailSize` to `SelectedFilesGrid` as the new `thumbnailSize:` parameter in the `.grid` case switch branch.
