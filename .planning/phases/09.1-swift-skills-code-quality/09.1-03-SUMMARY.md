# Phase 09.1 Plan 03: SwiftUI Accessibility and API Modernisation Summary

VoiceOver button traits, label, and hint added to the interactive map; inner Label hidden to prevent double-announcement; both empty states migrated to ContentUnavailableView; AppDesign.Radius tokens changed from CGSize to CGFloat across all 15 call sites; GeometryReader replaced with .onGeometryChange in CoordinateSearchPanel; four filter{}.count sites replaced with count(where:); Color.primary verbosity fixed; hard-coded padding replaced with design token.

## Decisions Made

- `AppDesign.Radius` tokens changed from `CGSize` to `CGFloat`; `RoundedRectangle(cornerSize:)` and `.clipShape(.rect(cornerSize:))` replaced with `cornerRadius:` variants at all 15 call sites atomically.
- `FileDropTargetModifier` internal stored property and extension method parameter renamed from `cornerSize: CGSize` to `cornerRadius: CGFloat`.
- `.onGeometryChange(for: CGFloat.self)` replaces the `GeometryReader` background block in `CoordinateSearchPanel`; consistent with existing usage in `FileIntakeView`.
- `count(where:)` is a Swift 6 `Sequence` method; replaces all four `filter{}.count` patterns in `FileIntakeViewModel`.
