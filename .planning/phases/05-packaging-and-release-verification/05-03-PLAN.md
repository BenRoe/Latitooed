<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Host environment to packaged app | Host may have Homebrew ExifTool, so smoke strips PATH to avoid false positives. |
| Packaged app to copied media files | Signed app mutates temporary copies only. |
| Signed app verification to public release claim | Signed `.app` verification does not imply notarized distribution. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-03-01 | Spoofing | external helper fallback | mitigate | Launch with stripped PATH and inspect metadata using app-bundled helper path. |
| T-05-03-02 | Tampering | source fixtures | mitigate | Copy fixtures to temp before writes; never mutate tracked fixtures. |
| T-05-03-03 | Repudiation | manual verification | mitigate | Capture expected/result evidence in `05-HUMAN-UAT.md`. |
| T-05-03-04 | Information Disclosure | release samples | mitigate | Use nonsensitive fixtures and temp working copies. |
| T-05-03-05 | Scope confusion | distribution readiness | mitigate | Docs explicitly defer notarization, stapling, DMG/ZIP, updater, and Mac App Store packaging. |
</threat_model>
