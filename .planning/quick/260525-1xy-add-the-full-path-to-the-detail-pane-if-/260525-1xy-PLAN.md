# Quick Task 260525-1xy: Improve Selected File Detail Pane

## Goal

When a single loaded file is selected, the detail pane should show the selected file's full path instead of only the containing folder name. The detail pane should also be collapsible so users can recover vertical space under the file list.

3. Make the detail pane collapsible with a native SwiftUI disclosure control.
4. Update focused view-model/smoke tests and run relevant checks when possible.
