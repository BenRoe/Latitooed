## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03-01 | Tampering | picker/drop URL handling | mitigate | Route all URLs through `FileIntakeService`; never add rows directly from UI event handlers. |
| T-01-03-02 | Denial of Service | bulk drop warnings | mitigate | Store latest warning details for the most recent event and avoid duplicate accepted rows. |
| T-01-03-03 | Information Disclosure | warning/detail UI state | mitigate | Show filename and containing folder rather than full path in primary row/detail copy. |
| T-01-03-SC | Tampering | package installs | accept | No package-manager installs are planned. |
</threat_model>
